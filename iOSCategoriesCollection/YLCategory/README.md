# HYCategory
###iOS开发中常用到的分类

1.	`UIView+Extension`：增加方便访问UIView位置的属性，例如直接使用`view.frame.origin.x`为`view.x`.
2.	`UIColor+Extension`：增加UIColor的方法，例如增加使用十六进制形式设置颜色的方法。
3.	`UIImage+Extension`：增加UIImage的方法，例如拉伸图片。
4.	`UIAlertView+Extension`：增加UIAlertView的方法，例如使用block代替selector创建UIAlertView.
5.	`UIImage+Gif`：一个使用Gif的UIImage的分类。
6.	`NSDate+Escort`：一个方便使用NSDate的分类，添加了许多操作日期的方法，例如：dateAtEndOfDay返回某天的最后一个时刻的NSDate对象。
7.	`NYXImagesKit`：<a href="https://github.com/Nyx0uf/NYXImagesKit" target="_blank">原作地址</a>增加UIImage的许多分类，包括设置图片大小，旋转图片，图片保存等。
8.	`BlicksKit`：<a href="https://github.com/zwaldowski/BlocksKit" target="_blank">原作地址</a> 为Objective-C的许多类增加了分类，使用方式都改为block实现，让代码可读性更强！
9.	`ObjectiveSugar`：<a href="https://github.com/supermarin/ObjectiveSugar" target="_blank">原作地址</a>对Foundation框架中的一些类，比如：NSArray，NSDictionary，NSString进行方法增强，使用的是ruby风格，非常好用。
10.	`NSObject+AutoDescription`：<a href="https://github.com/djmadcat/NSObject-AutoDescription" target="_blank">原作地址</a>只要在description中调用一个方法，就会自动生成改类的描述。在LLDB的时候特别有用。
